The Offer

Memorize the 1-sentence pitch

Say it 10 times. If you can say this confidently, you can sell.

“SojaAI helps local service businesses reply instantly to website inquiries so they stop missing bookings and convert more leads — automatically.”

What to say (simple outcomes)

  • Instant replies to website inquiries
  • Booking requests get priority alerts
  • All leads stay organized in one dashboard

What NOT to say (avoid tech words)

n8n Airtable webhooks OpenAI API
Rule: If you talk about results, the client understands. If you talk about tech, the client gets confused.

Plans (memorize)

Basic
  • $1,500 setup
  • $149/month
  • Best for: stop missing leads
Pro
  • $2,500 setup
  • $349/month
  • Best for: scale without hiring

Handoff line (memorize)

Call Flow

20-minute intro call (same structure every time)

Follow this flow. If you get lost, return to the structure.

0:00–2:00 Opening
  • Thank them for booking
  • Set the agenda
  • Ask permission for questions
2:00–8:00 Discovery
  • How leads come in
  • How fast they reply
  • What gets missed
8:00–13:00 Explain solution
  • Instant reply
  • Booking = hot alert
  • Dashboard for clarity
13:00–20:00 Price + next step
  • Recommend Basic/Pro
  • Handle 1 objection
  • Handoff to owner

Opening line (read exactly)

“Hi John, thanks for booking. This will be quick and focused. I’ll ask a few questions to see if this is a good fit — is that okay?”

Discovery questions (pick 5)

  • How do website inquiries come in today?
  • How fast do you usually reply?
  • Do you ever miss messages while busy?
  • How many inquiries do you get weekly?
  • What happens when someone asks for pricing?

Transition line

“Based on what you shared, faster replies would help you win more jobs. Let me explain a simple system — then you can tell me if it makes sense.”
Rule: Keep it calm. Ask questions. Do not rush.
Now you know the structure — don’t change it
Objections

Only 3 objections you must master

Do not learn 20 responses. Master these 3 first.

1) “It’s expensive”

“I understand. Most owners felt that at first. But missed bookings cost more than the system. If it saves time and wins more jobs, it pays for itself.”

2) “I need to think about it”

“Totally fair. What part do you need to think about — the result, or the investment?”

3) “Send me details”

“Sure. Before I send it, are you mainly interested in Basic or Pro? I’ll have our team email the official proposal and next steps.”
Rule: Handle one objection, then ask for next step. Don’t debate.

Price line (memorize)

“Basic is $1,500 setup + $149/month. Pro is $2,500 setup + $349/month. Based on what you shared, I recommend: (Basic/Pro).”

Close line (soft)

“If this makes sense, I can have our operations team send the official setup link today. Would you like us to proceed?”

Payment rule

If client asks for payment link:
“For security, payments are handled only through our official business email. I’ll have our team send the secure link today.”
If you master these 3, you’re ahead of most beginners
